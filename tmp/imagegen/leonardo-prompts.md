# Leonardo Blog Image Prompts

Use these prompts in the Leonardo web app to generate replacement blog images for the site.

## Recommended Leonardo setup

- Model: `Lucid Origin`
- Switch to `Lucid Realism` only if you want a more photo-like result
- Image type: landscape
- Size: `1536 x 1024` if available
- Count: `1` image per prompt
- Prompt Enhance: optional
- Public mode is fine on the free plan

## Global negative prompt

Use this for every image if Leonardo shows a negative prompt field:

`logos, trademarks, brand names, readable interface text, watermark, gibberish text, blurry details, cluttered composition, distorted hands, duplicate objects, warped screens, low contrast`

## File naming

After you download each final image, rename it to the exact filename below and place it in:

`output/imagegen/blog-batch/`

Then run:

```powershell
powershell -ExecutionPolicy Bypass -File .\tmp\imagegen\apply-blog-images.ps1
```

That script will crop and copy everything into:

`assets/images/blog/`
